FAQ

Frequently Asked Questions

Is my assessment data stored on a server? +
No — all data (profile, quiz answers, results) is stored exclusively in your browser's localStorage. Nothing is sent to any server. Clearing your browser data will remove your results.
Can I retake the assessment to track improvement? +
Yes! You can retake the assessment as many times as you like. Each retake overwrites the previous result. We recommend studying your weak topics for at least 2 weeks before retaking to see measurable improvement.
What domains will the system support in the future? +
The current assessment covers Computer Science fundamentals (programming, data structures, algorithms, problem solving, logical reasoning). Future versions aim to add domain-specific question banks for Data Science, AI/ML, Web Development, and Mathematics.
How are the skill gap levels determined? +
Each topic category is scored independently. Strong = ≥70% correct in that category, Moderate = 45–69%, Weak = <45%. These thresholds are based on standard educational mastery criteria. The recommendation engine then prioritises the weakest categories first.
Can this system be used in a classroom or institution? +
The current version is a research prototype. We welcome collaboration with educators and institutions interested in deploying or extending the system. Please contact us via the form above to discuss classroom integration or licensing.
Is the source code available? +
The system is built entirely with HTML5, CSS3, and Vanilla JavaScript — no frameworks or libraries. The code is open for academic review and collaboration. Contact us if you'd like access to the full source repository.
